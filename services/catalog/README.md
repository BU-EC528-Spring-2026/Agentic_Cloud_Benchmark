# Services

This directory holds service-level docs and metadata for benchmark targets.

Current service families:

- `billing_pricing/`
- `feature_router/`
- `maintenance_window/`
- `cache_api/`
- `queue_worker/`
- `payments_api/`
- `openclaw/`

Local and GitHub-derived fixture repositories live under `services/fixtures/`.

At the moment:

- local service families back the local task bank
- `openclaw/` backs the GitHub-derived task bank
