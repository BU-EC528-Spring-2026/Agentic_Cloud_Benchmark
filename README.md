# Agentic_Cloud_Benchmark
Agentic Cloud Benchmark (ACBench) is a standardized framework that evaluates autonomous AI agents on their ability to build, deploy, detect failures, and repair cloud systems using realistic operational scenarios from AIOPS Lab.

## Project Description
Link: https://drive.google.com/file/d/1Vch__BD6RJYEUNdN9Sptc4Tf5mXV88HE/view?usp=sharing 

## Project Structure
- `scenarios/` — benchmark task definitions
- `agents/` — AI agent logic and API integration
- `evaluator/` — scoring and result analysis
- `results/` — output logs from benchmark runs

## Setup
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Add your API keys to a `.env` file:
```
OPENAI_API_KEY=your_key_here
ANTHROPIC_API_KEY=your_key_here
```
