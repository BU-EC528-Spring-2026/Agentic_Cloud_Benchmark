# Agentic_Cloud_Benchmark
Agentic Cloud Benchmark (ACBench) is a standardized framework that evaluates autonomous AI agents on their ability to build, deploy, detect failures, and repair cloud systems using realistic operational scenarios from AIOPS Lab.
