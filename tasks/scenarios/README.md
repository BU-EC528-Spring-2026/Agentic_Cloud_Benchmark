# Scenarios

This directory contains the canonical benchmark task definitions.

Scenario sources:

- `local/`
- `github/`

All supported scenario assets for this repo now live under `tasks/scenarios/`.

Under each source directory, scenarios are grouped by mode:

- `code/`
- `ops/`
- `combined/`

Before adding a new scenario, read:

- `docs/TASK_BANK_REQUIREMENTS.md`
- `docs/SCENARIO_AUTHORING.md`
